How it works

This tool replays a CSV record back into the HSMI app — no data re-entry needed.

  1. Upload or paste your CSV — any HSMI mobility export, single-row or batch.
  2. Select the member row — identified by name, date, and location.
  3. Click "Open Report" — the app opens in a new tab with all fields pre-filled and results calculated.
  4. Print to PDF — use the app's Print button (or Ctrl/Cmd+P) to generate the report as you normally would.

This file lives at /hsmi-regenerate/index.html and loads the HSMI app from /mobility-age/index.html. Both must be deployed on the same Netlify site.
